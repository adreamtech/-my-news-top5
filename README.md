# 热点视频聚合 App（PWA）

一个可在手机浏览器中安装的轻量应用，用于聚合中国主流视频平台热点视频。

## 功能

- 聚合展示抖音、B站、快手、微博、小红书热点视频条目（演示数据）。
- 支持按平台筛选。
- 支持关键词一键打开多平台搜索。
- 支持 PWA 安装（“添加到主屏幕”）与离线缓存。

## 运行方式

由于 Service Worker 需要 HTTP 环境，请使用本地服务器启动：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080`

## 安装方式（手机）

1. 用 Chrome/Safari 打开页面。
2. Android Chrome 点击“安装 App”或浏览器菜单中的“安装应用”。
3. iOS Safari 点击“分享” -> “添加到主屏幕”。
